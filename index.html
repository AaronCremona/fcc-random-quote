About Lyrical Lunacy

This website is a project I built for Free Code Camp's Random Quote Zipline. I used Javascript and JQuery. I spent way too long listening to Spotify picking out songs to use for the app, but had a lot of fun doing it. I probably should have moved on to more Javascript instead!

Under the hood, I created an object to contain all the quotes and methods needed to retrieve the track preview from Spotify, images, and all the artist / track info. The only information I have stored in the object is the quote itself and the spotify ID for the track. Everything else is dynamically loaded. The app calculates the number of quotes available on startup, so it is easily modified. I was really thinking about diving into React right away for the Ziplines, but in the end I decided it would be better to get some more practice with the basics. I plan to work on beefing up my object oriented skill during the rest of the ziplines to have a better foundation to add React and backend technologies.

The site design was heavily influenced (read: *cough cough* ripped off *cough*) from Spotify's individual track landing pages. I really thought the blurred background album art was pretty sweet.
